Library Book Organizer is an interactive web-based game built using HTML, CSS, and JavaScript to help users understand the concept of the Insertion Sort algorithm in a simple and visual way.

In this game, books with numbered spines are placed randomly on a shelf. The player’s task is to organize them in ascending order by inserting each book into its correct position, just like how insertion sort works. As the game progresses, users can visually see how elements are compared, shifted, and inserted step-by-step.

✨ Features:

* Interactive gameplay with drag-and-drop or click-based insertion
* Real-time visualization of insertion sort operations
* Highlighting of sorted and unsorted sections
* Smooth animations for shifting and comparisons
* Step-by-step explanation of the algorithm
* Move counter and timer for performance tracking
* Multiple difficulty levels (Easy, Medium, Hard)
* Auto-solve mode to demonstrate the algorithm

🎯 Objective:
The goal of this project is to make learning sorting algorithms more engaging by turning them into an interactive experience instead of static code or theory.

🛠️ Tech Stack:

* HTML
* CSS
* JavaScript (Vanilla)

📈 Algorithm Insight:
Insertion Sort works by taking one element at a time and inserting it into its correct position within the already sorted portion of the list.

* Best Case: O(n)
* Average Case: O(n²)
* Worst Case: O(n²)

This project is ideal for beginners who want to visualize and understand how insertion sort operates internally.
